# Rule-Based Chatbot (Python)

## Overview
This is a simple Rule-Based Chatbot developed using Python. The chatbot responds to user messages based on predefined rules, patterns, and responses.

## Features
- Greeting responses
- Goodbye responses
- Thank-you responses
- Answers simple questions
- Uses regular expressions for pattern matching
- Random response selection for natural conversations

## Technologies Used
- Python 3
- Random Module
- Regular Expressions (re)
